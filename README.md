# データ形式
## kansoku[year].csv
```
地震ID(int), 緯度(float), 経度(float), 計測震度(float), 震度階級(int, 0 ~ 9)
```
## shingen[year].csv
```
地震ID(int), 観測点数(int), 緯度(float), 経度(float), 深さ(float), 気象庁マグニチュード(float)
```
